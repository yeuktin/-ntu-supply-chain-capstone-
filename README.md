# Predicting Late Deliveries and Optimizing Delivery Prioritization in Supply Chains

Machine learning capstone project completed at Nanyang Technological University (NTU) focused on predicting late delivery risk using large-scale supply chain transaction data.
## Project Overview

This project analyzes over 180,000 supply chain transactions to identify operational factors contributing to delivery delays and develop predictive models for late delivery risk.

The goal of the project was to support logistics decision-making and improve delivery prioritization through data-driven insights and machine learning techniques.

---

## Key Features

- Exploratory Data Analysis (EDA)
- Data cleaning and preprocessing
- Classification modeling for delivery risk prediction
- Supply chain performance analysis
- Business-focused operational insights

---

## Tools & Technologies

- Python
- pandas
- scikit-learn
- matplotlib

---

## Results

- Identified shipping methods with significantly higher late delivery risk
- Analyzed regional and operational delivery bottlenecks
- Developed predictive models to support logistics prioritization decisions
- Presented findings in a capstone defense at NTU Singapore
